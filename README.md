# David's Homebrew Tap

Install apps from this tap using Homebrew.

## Add this tap

```bash
brew tap jimezsa/tap
```

## Install apps

If you already tapped it, install `jobcli` with:

```bash
brew install jobcli
```

Or tap + install in one command:

```bash
brew install jimezsa/tap/jobcli
```

## Update and upgrade

```bash
brew update
brew upgrade
```

## Discover available apps in this tap

```bash
brew search /jimezsa/tap/
```
